Two types of arrays

1.Single dimesional 
2.two/Multi dimensional

Single Dimensional
1. Declare an Array
2. insert values into array
3.Find size of an array
4.Read values from an array

syntax:
int a[] = new int[5];

a[0]=100;
a[1]=200;
a[2]=300;
a[3]=400;

 
 if we dont know the exact size of array use below syntax
 
 int a[]={100,200,300,400,500,600};
 
 
2.two/Multi dimensional
1. Declare an Array
2. insert values into array
3.Find size of an array
4.Read values from an array

int a[][]=new int[3][2];

a[0][0]=100;
a[0][1]=200;
a[1][0]=30;
a[1][1]=40;
a[2][0]=50;
a[2][1]=60;

if we dont know the exact size of array use below syntax

int a[][]={{100,200},{200,300},{400,500}};
